    22nd August 2022
  • Today's training session was about Custom labels, Static Resources & Navigations
  • Custom labels enable developers to create multilingual applications by automatically presenting information (for example, help text or error messages) in a user's native language
  • Static resources allow us to upload content that you can reference in a Visualforce page, including archives (such as . zip and . jar files), images, style sheets, JavaScript, and other files. These can be used only in Salesforce Org.
  • A lifecycle hook is a callback method triggered at a specific phase of a component instance’s lifecycle.
  • constructor() is called when the component is created & which calls before the connectedCallback().
  • connectedCallback() which is used to interact with a component's environment
  • renderedCallback() is called after every render of the component.
  • render() is called to update the UI.
  • disconnectedCallback() is called when the element is removed from a document & this hook flows from parent to child.
  • errorCallback(error, stack) is called when a descendant component throws an error. The error argument is a JavaScript native error object, and the stack argument is a string.
  • The lightning/navigation service is supported only in Lightning Experience, Lightning Communities, and the Salesforce app.
